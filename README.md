# Confusable Characters Detector

A visual tool to detect and analyze confusable/lookalike characters in strings, hashes, and tokens.

- **Character Type Detection** - Identifies letters, numbers, symbols, and spaces
- **Ambiguous Character Highlighting** - Detects confusable characters like `i`, `1`, `l`, `L` and `O`, `0`
- **Color-Coded Visualization** - Easy-to-read visual representation of each character type
- **Real-time Statistics** - Live character count breakdown
